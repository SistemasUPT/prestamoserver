# Sistema de Préstamo de Servidores - UPT

Sistema web para gestión de préstamos de servidores de la Universidad Privada de Tacna.

## Acceso
- **URL Principal**: https://sistemasupt.github.io/prestamoserver/
- **Apps Script**: https://script.google.com/macros/s/TU_SCRIPT_ID/exec

## Funcionalidades
- Registro de solicitudes de préstamo
- Gestión de servidores disponibles
- Autorización de soporte técnico
- Reportes y estadísticas
